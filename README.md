# Игра Сапёр

Online пример: [https://ipatov.github.io/sapper/](https://ipatov.github.io/sapper/)

## Запуск и сборка
#### Установка
```
git clone git@github.com:Ipatov/sapper.git
cd sapper
npm install
```

#### Старт dev
```
npm run start
```

#### Сборка 
```
npm run build
```

#### Тесты
```
npm run test
```
